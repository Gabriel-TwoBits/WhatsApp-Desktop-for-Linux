# WhatsApp-Desktop-for-Linux
a simple script for installing a WhatsApp Web container on Linux. no stores dependecies, only local installing. made for Debian-base and Arch-base users.

# Português
Para realizar a instalação, apenas baixe o script adequado pra você e o execute. Se quiser fazer pela interface gráfica, basta apertar o botão direito > Propriedades > Permitir execução como programa. Depois botão direito > Executar como um programa.
Caso prefira por terminal, cd diretorio/com/o/arquivo > chmod +x nome_do_arquivo > ./nome_do_arquivo

O mesmo processo serve para o script de desinstalação. Note que ele NÃO vai desinstalar o nodejs e o npm, os quais devem ser removido manualmente pelo usuário, caso queira.

Sinta-se à vontade para alterar o script e diretórios de acordo com suas preferências.

Esse script vem como uma alternativa para os diversos WhatsApp for Linux que existem nas lojas, dos quais nenhum funcionou adequadamente pra mim. Essa instalação local através do script tem apresentado um resultado bem mais satisfatório, tanto em quesito de funcionalidade quanto no consumo de recursos.

# English
To perform the installation, simply download the appropriate script for you and run it. If you prefer to do it through the graphical interface, just right-click > Properties > Allow executing file as program. Then right-click > Run as a program.

If you prefer using the terminal, run cd directory/with/the/file > chmod +x file_name > ./file_name.

The same process applies to the uninstallation script. Note that it will NOT uninstall nodejs and npm, which must be removed manually by the user if desired.

Feel free to modify the script and directories according to your preferences.

This script comes as an alternative to the numerous WhatsApp for Linux applications available in the stores, none of which worked adequately for me. This local installation via the script has delivered a much more satisfactory result, both in terms of functionality and resource consumption.
